## Опис реалізації

Програма реалізована на мові **C#** і виконує пошук симетричних слів (паліндромів) у текстовому масиві.

### 1. Алгоритм пошуку паліндрома

Для перевірки слова використовується метод `IsPalindrome`, що реалізує алгоритм **"двох вказівників"**. Це дозволяє мінімізувати кількість операцій, порівнюючи символи з початку та кінця одночасно:

* Всі символи зводяться до нижнього регістру (`ToLower`).
* Слово вважається паліндромом, якщо воно читається однаково в обох напрямках і має довжину більше 1 символу.

### 2. Обробка тексту

Метод `FindLongestPalindrome` виконує наступні кроки:

* **Токенізація:** Розбиття вхідного тексту на масив слів за допомогою роздільників (пробіли та знаки пунктуації).
* **Фільтрація:** Перевірка кожного слова на відповідність критеріям паліндрома.
* **Порівняння:** Пошук слова з максимальною довжиною (`Length`).

### 3. Модульне тестування (Unit Testing)

Для перевірки надійності коду використано бібліотеку **NUnit**. Тести охоплюють:

* Пошук найдовшого слова серед декількох варіантів.
* Регістронезалежність (наприклад, "Радар" — валідний паліндром).
* Відсікання занадто коротких слів (однолітерні слова).
